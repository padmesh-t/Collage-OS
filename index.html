Inter-Process Communication (IPC) Using User-Defined Signals

Explanation:
This program demonstrates how a parent and child process communicate using user-defined signals (SIGUSR1 and SIGUSR2). The parent sends signals to the child, which responds by performing tasks and then exiting. This simulates real OS-level process communication.

Implementation of Buddy System Memory Allocation

Explanation:
This program simulates dynamic memory allocation using the Buddy System algorithm. You can allocate and deallocate memory blocks, and the system merges adjacent free blocks automatically.
