INTER-PROCESS COMMUNICATION (IPC) USING USER-DEFINED SIGNALS

Objective: To demonstrate how processes communicate in Linux using custom-defined signals.

Concepts Used: Signals, process communication, system calls like kill() and signal().

Outcome: Understanding process synchronization and signal handling mechanisms in operating systems.

Implementation of Buddy System Memory Allocation

Objective: To implement and visualize memory allocation using the Buddy System algorithm.

Concepts Used: Memory management, binary tree partitioning, allocation and deallocation tracking.

Outcome: Efficient management of dynamic memory and reduction of fragmentation.
